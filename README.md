## 📘 About xmum-beamer-template

This is the deep blue theme Beamer template designed for Xiamen University Malaysia (XMUM). It helps you create clean, professional presentation slides using LaTeX with an attractive, calm blue style.

The template includes ready-to-use slide designs. It works well for students and faculty who want to save time on slide formatting and focus on content. 

You do not need special skills. You only need a basic LaTeX setup and this template to get started.

## 💻 System Requirements

- **Operating System:** Windows 10 or later  
- **Software Needed:**  
  - LaTeX distribution such as MiKTeX (recommended for Windows)  
  - A text editor or LaTeX editor (e.g., TeXworks, TeXstudio, or VS Code with LaTeX extension)  
- **Disk Space:** Around 100 MB free space for LaTeX and template files  

If you do not have LaTeX installed, the setup section will guide you on how to get it.

### Step 2: Get the Files

1. On the GitHub page, find the green “Code” button near the top right.  
2. Click it and then click “Download ZIP.”  
3. Save the ZIP file to your computer (e.g., Desktop or Downloads folder).  
4. Right-click the ZIP file and select “Extract All” to unzip the files into a folder.

### Step 3: Install LaTeX

You must have LaTeX on your PC to use the template. Follow these steps:

1. Go to the MiKTeX website: https://miktex.org/download  
2. Click the “Download” button for Windows.  
3. Run the downloaded installer file and follow the instructions. Keep the default settings.  
4. After installation, open the MiKTeX console and update the package database if prompted.

Alternatively, install another LaTeX distribution if you prefer.

### Step 4: Open the Template

Inside the extracted folder, you will find a `.tex` file. This is the main slide file.

- Use an editor like TeXworks or TeXstudio to open this `.tex` file.  
- If you don’t have an editor, TeXworks usually comes with MiKTeX. You can open it from the Start menu after installing MiKTeX.

### Step 5: Build Your Presentation

1. Open the `.tex` file in your editor.  
2. Press the 'Build' or 'Compile' button (often shown as a green arrow).  
3. This will generate a PDF file with your slides.  

If errors occur during compiling, check you installed MiKTeX correctly and all packages are updated.

### Step 6: Edit Your Slides

You can change the text inside the `.tex` file to add your own content.

- Look for sections marked “%” with notes or instructions.  
- Replace placeholder text with your information.  
- Save the file and compile again to see changes.

## ❓ Troubleshooting Tips

- If the presentation does not compile, check if MiKTeX is installed and updated.  
- If the PDF file does not open, make sure you have a PDF reader installed.  
- Use the MiKTeX console to refresh package database and install missing packages automatically.  
- Avoid changing file names or folder paths in the project.
